# NetworkComplaints
App Where You Register Network Issues

# Prerequisites
1. Java<br>
2. Android Studio<br>
3. Android SDK

# Installing
1. Download or Clone the Project. <br>
2. Build the project with Android Studio.

# Built With
Java<br>
Android SDK<br>
<a href="https://loopj.com/android-async-http/">Android Async Http Client</a>
